## Setup and Run

For this project, we make use of <b>yarn</b>. Assuming that you have those pre installed on your machine, follow these steps:
<br>

- Run yarn to install the project dependencies:

```bash
yarn install
```

<br>

- Run (Android):

```bash
yarn android
```

<br>

- Run (iOS):

```bash
cd ios
pod install
cd ..

yarn ios
```
